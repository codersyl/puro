# **MIME类型**
媒体类型（通常称为 Multipurpose Internet Mail Extensions 或 MIME 类型 ）是一种标准，用来表示文档、文件或字节流的性质和格式。被HTTP协议采用

格式：type/subtype，由类型与子类型两个字符串中间用'/'分隔而组成。不允许空格存在。

七大类别
* text
* image
* audio
* video
* application
* multipart
* message
